Problem F: 快速排序

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 1637    Accepted: 551    Creator: Imported
Problem Description
著名的快速排序算法里有一个经典的划分过程：我们通常采用某种方法取一个元素作为主元，通过交换，把比主元小的元素放到它的左边，比主元大的元素放到它的右边。 
使用快速排序算法编写程序，将输入的若干整数，按照从小到大的顺序排序并输出。
快速排序算法描述：
快速排序算法是基于分治策略的，对于数组a[p..r]，进行分治法设计：
分解：数组a[p..r]被划分成两个（可能空）的子数组a[p..q-1] 和 a[q+1..r]，使得：A[p..q-1] ≤ A[q]、 A[q+1..r] ≥ A[q]。下标q在划分阶段进行计算。
求解：递归调用快速排序，对两个子数组A[p..q-1] 和 A[q+1..r]进行排序。
合并：因为两个子数组是就地排序的，将它们合并并不需要操作，整个数组A[p..r]已排序。



Input Description
第1行输入一个正整数 N （n <= 1000000）； 
第2行是空格分隔的 N 个不同的整数，每个整数不超过 int 表示的范围。
Output Description
按照从小到大的顺序输出 N 个整数，用 1 个空格分隔。注意：最后一个整数后没有空格。
Sample Input
5
1 3 4 2 5
Sample Output
1 2 3 4 5
