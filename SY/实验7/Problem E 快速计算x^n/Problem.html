Problem E: 快速计算x^n

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 1630    Accepted: 1025    Creator: Imported
Problem Description
我们注意到，如果 n 是 2 的幂，则可以通过自乘的方法计算 x^n。例如：x^4 是 x^2 的平方，所以 x^4 可以用2次乘法计算，而不需要3次乘法。
对于任意的正整数 n，如果 n 是偶数，则 x^n=(x^(n/2))^2；如果 n 是奇数，则 x^n=x*x^(n-1)。
编写计算 x^n 的递归函数 power（递归在 n=0 时结束，此时函数返回1）。
编写一个程序，输入 x 和 n 的值，调用 power 函数计算 x^n，然后显示函数的返回值。
测试程序为：
#include <stdio.h>
int power(int x, int n);
int main()
{
    int x, n;
    scanf("%d %d", &x, &n);
    printf("%d\n", power(x, n));
    return 0;
}
/* 你的代码将被嵌在这里 */
Input Description
在一行中输入两个整数 x 和 n (0 <= n，且保证 x^n 在 int 范围内)。
Output Description
输出所求的结果。
Sample Input
2 5
Sample Output
32
