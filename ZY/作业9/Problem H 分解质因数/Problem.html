Problem H: 分解质因数

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 616    Accepted: 311    Creator: Imported
Problem Description
每个非素数（合数）都可以写成几个素数（也可称为质数）相乘的形式，这几个素数就都叫做这个合数的质因数。比如，6可以被分解为2*3，而24可以被分解为2*2*2*3。
现在，你的程序要读入一个[2,100000]范围内的整数，然后输出它的质因数分解式；当读到的就是素数时，输出它本身。
Input Description

一个整数，范围在[2,100000]内。

Output Description
形如：n=a*b*c*d  或  n=n
所有的符号之间都没有空格。abcd这样的数字一定是从小到大排列的。
Sample Input
18
Sample Output
18=2*3*3
