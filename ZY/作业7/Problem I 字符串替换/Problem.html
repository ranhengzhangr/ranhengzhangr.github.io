Problem I: 字符串替换（15分）

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 515    Accepted: 344    Creator: admin
Problem Description

本题要求编写程序，将给定字符串中的大写英文字母按以下对应规则替换：

原字母 对应字母
A Z
B Y
C X
D W
… …
X C
Y B
Z A
Input Description
输入在一行中给出一个不超过80个字符、并以回车结束的字符串。
Output Description
输出在一行中给出替换完成后的字符串。
Sample Input
Only the 11 CAPItaL LeTtERS are replaced.
Sample Output
Lnly the 11 XZKRtaO OeGtVIH are replaced.
