Problem A: 跟奥巴马一起编程

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 669    Accepted: 188    Creator: Imported
Problem Description

美国总统奥巴马不仅呼吁所有人都学习编程，甚至以身作则编写代码，成为美国历史上首位编写计算机代码的总统。2014年底，为庆祝“计算机科学教育周”正式启动，奥巴马编写了很简单的计算机代码：在屏幕上画一个正方形。现在你也跟他一起画吧！

Input Description

输入在一行中给出正方形边长N（3<=N<=20）和组成正方形边的某种字符C，间隔一个空格。

Output Description

输出由给定字符C画出的正方形。但是注意到行间距比列间距大，所以为了让结果看上去更像正方形，我们输出的行数实际上是列数的50%（取整）。

Sample Input
10 a
Sample Output
aaaaaaaaaa
a        a
a        a
a        a
aaaaaaaaaa
