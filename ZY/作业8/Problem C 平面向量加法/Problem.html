Problem C: 平面向量加法（15 分）

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 523    Accepted: 335    Creator: admin
Problem Description
本题要求编写程序，计算两个二维平面向量的和向量。
Input Description
输入在一行中按照“x1 y1  x2 y2 ”的格式给出两个二维平面向量v1=(x1,y1) 和v2=(x2,y2)的分量。
Output Description
在一行中按照(x, y) 的格式输出和向量，坐标输出小数点后一位（注意不能输出−0.0 ）。
Sample Input
3.5 -2.7 -13.9 8.7
Sample Output
(-10.4, 6.0)
