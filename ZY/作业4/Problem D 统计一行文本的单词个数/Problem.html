Problem D: 统计一行文本的单词个数（15 分）

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 659    Accepted: 405    Creator: Imported
Problem Description

本题目要求编写程序统计一行字符中单词的个数。所谓“单词”是指连续不含空格的字符串，各单词之间用空格分隔，空格数可以是多个。

Input Description

输入给出一行字符。

Output Description
在一行中输出单词个数。
Sample Input
Let's go to room 209.
Sample Output
5
