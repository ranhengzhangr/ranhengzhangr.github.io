Problem H: 最大公约数和最小公倍数

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 933    Accepted: 507    Creator: admin
Problem Description
编写程序求两个给定正整数的最大公约数和最小公倍数。
Input Description

在一行中输入两个正整数 M 和 N，其中 M 和 N <= 10^4。

Output Description
在一行中顺序输出 M 和 N 的最大公约数和最小公倍数，两数字间以1个空格分隔。
Sample Input
2 3
Sample Output
1 6
