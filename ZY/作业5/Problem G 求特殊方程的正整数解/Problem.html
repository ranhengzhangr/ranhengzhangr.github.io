Problem G: 求特殊方程的正整数解（15 分）

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 944    Accepted: 231    Creator: admin
Problem Description
本题要求对任意给定的正整数N，求方程X^2+Y^2=N 的全部非负整数解。
Input Description

输入在一行中给出正整数N （≤10000）。

Output Description
输出方程X^2+Y^2=N的全部非负整数解，其中X≤Y 。每组解占1行，两数字间以1空格分隔，按X的递增顺序输出。如果没有解，则输出No Solution 。
Sample Input
884
Sample Output
10 28
20 22
