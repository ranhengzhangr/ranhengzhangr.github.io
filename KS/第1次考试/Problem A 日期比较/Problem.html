Problem A: 日期比较

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 603    Accepted: 198    Creator: admin
Problem Description
给出两个日期date1和date2，请你编写程序，比较两个日期的大小。
Input Description
第一行按照格式month/day/year输入日期date1。
第二行按照格式month/day/year输入日期date2。
题目保证输入日期的有效性。
Output Description
如果date1大于date2，则输出1；如果date1小于date2，则输出-1；如果date1等于date2，则输出0。
Sample Input
12/16/2019
1/6/2020
Sample Output
-1
