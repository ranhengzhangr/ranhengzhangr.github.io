Problem B: 二分法求解根号2

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 673    Accepted: 184    Creator: admin
Problem Description
首先介绍二分法求解根号2的近似值的方法，然后请你按照描述的算法编程求解。
设2^(1/2)=a，则a^2=2
由于函数f(x)=x^2在[1,2]上是严格单调递增的，因此可以采用二分法来逼近根号2的值。
设 精度为eps，left=1，right=2，根据left和right的中点mid处的函数值f(mid)与2的大小来选择子区间进行逼近：
(1)如果f(mid)>2，说明mid大于根号2，则在[left,mid]区间内继续逼近，因此让right=mid
(2)如果f(mid)<2，说明mid小于根号2，则在[mid,right]区间内继续逼近，因此让left=mid
当right-left<eps时结束，mid即为所求的近似值。
Input Description
输入一个精度。
Output Description
输出根号2的近似值，保留6位小数。
Sample Input
1e-5
Sample Output
1.414207
