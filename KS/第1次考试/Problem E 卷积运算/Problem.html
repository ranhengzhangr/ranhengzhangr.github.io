Input Description
第一行输入一个整数n，表示矩阵f为n*n矩阵（3<=n<=32）。
然后是n行，每行n个整数，以空格分隔。
再输入一个整数m，表示矩阵g为m*m矩阵（2<=m<=5）。
然后是m行，每行m个整数 ，以空格分隔。
Output Description
输出卷积运算的结果，详见样例输出。
Sample Input
4
1 2 3 0
0 1 2 3
3 0 1 2
2 3 0 1
3
2 0 1
0 1 2
1 0 2

6 15
