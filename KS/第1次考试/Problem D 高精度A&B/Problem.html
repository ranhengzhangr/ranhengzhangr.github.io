Problem D: 高精度A+B

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 482    Accepted: 32    Creator: admin
Problem Description
对一道A+B的题目，如果A和B的范围在int范围内，相信大家很快就可以写出程序，但是如果A和B是有1000个数位的整数，恐怕就没有办法使用已有的数据类型来表示了，这时只能去模拟加法的过程。怎么样？听起来像是小学生学的东西吧！实际上原理就是小学的，所以不要去害怕这个看上去很高深的东西。
大整数又称为高精度整数，其含义就是用基本数据类型无法存储其精度的整数。
Input Description
输入为两行，每行为一个不超过1000个数位的整数。每个整数都是一个正数，并且保证不包含前缀零。
Output Description
输出两个整数的和。
Sample Input
11111111111111111111
22222222222222222222
Sample Output
33333333333333333333
