Problem K: 单词查找

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 358    Accepted: 128    Creator: admin
Problem Description
题目很简单，就是编写程序在一篇英文文章中查找指定的单词，如果找到，输出YES，否则输出NO。
Input Description

多组测试数据。

第一行输入一个正整数N（<100），代表后面有N组测试数据。

接下来输入N组测试数据，每组占两行：

第一行输入一篇英文文章，不超过1000个字符。

第二行输入一个单词，不超过10个字符。

Output Description
对每组测试数据，在一行中输出所求的结果。
Sample Input
2
To C, or not to c: that is the question.
that
how are you!
your
Sample Output
YES
NO
Hint
输入一组，就处理一组。不是把所有N组全部输入了，再处理。
