Problem C: 判断素数

Time Limit: 1 Sec    Memory Limit: 12 MB
Total Submissions: 2863    Accepted: 520    Creator: Imported
Problem Description
题目很简单，给你一个整数，判断是不是素数，如果是，输出YES，否则输出NO。
Input Description
多组测试数据。每组一行(<10^9)，当输入-1时结束。
Output Description
对每组测试数据，在一行中输出YES或NO。
Sample Input
2
3
9
-1
Sample Output
YES
YES
NO
