Problem D: 整数算术运算

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 771    Accepted: 308    Creator: admin
Problem Description
本题要求编写程序，计算2个正整数的和、差、积、商并输出。题目保证输入和输出全部在整型范围内。
Input Description
输入在一行中给出2个正整数A和B，以空格分隔。
Output Description
在5行中按照格式“A 运算符 B = 结果”顺序输出和、差、积、商。
Sample Input
3 2
Sample Output
3 + 2 = 5
3 - 2 = 1
3 * 2 = 6
3 / 2 = 1
