Problem H: IP地址转换

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 735    Accepted: 522    Creator: admin
Problem Description
一个IP地址是用四个字节（每个字节8个位）的二进制码组成。请将32位二进制码表示的IP地址转换为十进制格式表示的IP地址输出。
Input Description
输入在一行中给出32位二进制字符串。
Output Description
在一行中输出十进制格式的IP地址，其由4个十进制数组成（分别对应4个8位的二进制数），中间用“.”分隔开。
Sample Input
11001100100101000001010101110010
Sample Output
204.148.21.114
Hint
二进制转换为十进制方法：1111=((1*2+1)*2+1)*2+1=15
