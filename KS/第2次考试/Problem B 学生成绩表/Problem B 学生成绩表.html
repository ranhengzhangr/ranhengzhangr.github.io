Problem B: 学生成绩表

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 618    Accepted: 187    Creator: Imported
Problem Description

C语言课程今天考试结束后，老师要计算出你的期评成绩，并打印出成绩表上交给学院教学科，评定你的课程学习是否合格。努力做，免得重修啊。

输入一个班n名学生的姓名、学号、作业成绩、实验成绩、期末考试成绩，按照公式“期评成绩=作业成绩*10%+实验成绩*30%+期末考试成绩*60%”进行计算，结果保留2位小数。请你编写程序帮忙计算，并输出成绩表。

Input Description
第1行：正整数n， n<100
第2行：第1个学生的姓名 学号 作业成绩 实验成绩 期末考试成绩
第3行：第2个学生的姓名 学号 作业成绩 实验成绩 期末考试成绩
... ... ...

第n+1行：第n个学生的姓名 学号 作业成绩 实验成绩 期末考试成绩

其中姓名和学号均为不超过10个字符的字符串，成绩为0到100之间的一个整数，相互之间以空格分隔。

Output Description

输出n行，每行的输出格式为：

姓名 学号 作业成绩 实验成绩 期末考试成绩 期评成绩

注意：相互之间以1个空格分隔，行末没有空格。

Sample Input
3
zhangsan 031740188 80 70 65
lisi 031740199 86 95 88
wangwu 031740177 56 75 60
Sample Output
zhangsan 031740188 80 70 65 68.00
lisi 031740199 86 95 88 89.90
wangwu 031740177 56 75 60 64.10
