Problem I: 矩阵的快速幂

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 234    Accepted: 108    Creator: tanmj
Problem Description
设A是一个n×n的矩阵，求A^k的过程称为矩阵的快速幂是指：A^k = A^(k/2)*A^(k/2)*A^(k%2)。例如：A^8=A^4*A^4，A^7=A^4*A^2*A
Input Description
第一行输入一个整数n，表示后面有一个n*n的矩阵A，2<=n<=3。
然后是n行，每行n个整数，每个整数属于(-10, 10)，以空格分隔。
最后一行是一个整数K(0<=k<=30)。
Output Description
输出矩阵A的k次幂。
Sample Input
2
0 1
1 1
10
Sample Output
34 55
55 89
