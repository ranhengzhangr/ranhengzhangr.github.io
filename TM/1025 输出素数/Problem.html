1025: 输出素数

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 800    Accepted: 253    Creator: Imported
Problem Description
输入两个正整数a、b，找出[a, b]之间的素数，由小到大输出。(2 <= a <= b <=1000000)
Input Description

每一行输入两个正整数，两数之间有一个空格，可能有多行输入!

tips：利用while(scanf("%d %d", &a, &b)==2)语句实现循环输入变量

Output Description
输出[a,b]之间的素数，素数之间用一个空格隔开，每个输出占一行
Sample Input
2 5
3 11
Sample Output
2 3 5
3 5 7 11
