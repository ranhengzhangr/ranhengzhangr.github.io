1016: 鞍点

Time Limit: 1 Sec    Memory Limit: 128 MB
Total Submissions: 184    Accepted: 105    Creator: Imported
Problem Description
给定一个n*n矩阵A。矩阵A的鞍点是一个位置（i，j），在该位置上的元素是第 i 行上的最大数，第 j 列上的最小数。一个矩阵A也可能没有鞍点。你的任务是找出A的鞍点。
Input Description
输入的第 1 行是一个正整数n, （1<=n<=100），然后有n行，每一行有n个整数，同一行上两个整数之间有一个或多个空格。
Output Description
对输入的矩阵，如果找到鞍点，就输出其下标。下标为两个数字，第一个数字是行号，第二个数字是列号，均从0开始计数。
如果找不到，就输出：NO
题目所给的数据保证了不会出现多个鞍点。
Sample Input
4 
1 7 4 1 
4 8 3 6 
1 6 1 2 
0 7 8 9
Sample Output
2 1
